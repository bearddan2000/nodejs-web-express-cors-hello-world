# nodejs-web-express-cors-hello-world

## Description
A POC web sites using CORS
to call each others images.

## Tech stack
- nodejs
  - express
  - cors

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost
- http://localhost:81

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
